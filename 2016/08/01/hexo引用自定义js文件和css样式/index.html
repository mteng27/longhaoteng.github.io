hexo引用自定义js文件和css样式

发表于2016-08-01   |   分类于 hexo   |     |   阅读次数

博主用的是next主题，别的主题目录结构可能不太一样，但是整个框架是一样的，生成方式是一样的，所以引用方式也是相同的

添加自定义js

  • 首先把js文件放在\themes\next\source\js\src文件目录下

  • 然后找到\themes\next\layout目录下的布局文件

  • 把script引用代码加入到该文件中即可

<script type="text/javascript" src="/js/src/js文件名.js"></script>

添加自定义css样式

  • 添加外部css样式目的无非是想改变主题默认标签属性，可以使用浏览器的开发者或者检查功能找到相应id选择器名称或者class选择器名称，然后进行修改，这里注意在属性后面一定要加上!important

  • 同理，添加外部css样式和引用自定义js代码是一样的，在对应css文件夹内添加自定义外部css样式文件，然后在layout文件中添加引用即可

小小的一份支持，大大的一份鼓励！
